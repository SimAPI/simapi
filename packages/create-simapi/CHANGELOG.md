# create-simapi

## 0.0.1

### Initial release

- Scaffold a new SimAPI project with `npm create simapi@latest <name>`
- Interactive prompts for project name, description, auth handler, debug console, and Dockerfile
- Generates `endpoints/`, `simapi.config.ts`, `package.json`, `tsconfig.json`, `.gitignore`
- Optionally generates `authHandler.ts` and `Dockerfile`
